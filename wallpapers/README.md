## Sourcing plan

For v1.0 alpha:
- Cyberpunk: pull permissively-licensed art from Wallhaven/Unsplash that matches palette (purple/cyan/magenta).
- Abstract gradients: render programmatically at 4K + add subtle film grain.
- Geometric: programmatic from `aurora` palette with shifted parameters.
- Dark photo: source from Unsplash (CC0 photographers like Joel Filipe / Pawel Czerwinski).

For v1.0 final:
- Commission or generate via Midjourney / Stable Diffusion / Flux at 5K minimum.
